# 创意绘画板

## 目标

- 给定底图，底图内有浅色的块状组成。 用户可自行选择颜色 填充到底图的某区域中
- 底图分难易程度， 和不同分类
- 除颜色外 支持线条轮廓。 可在底图色块填充边界线颜色； 或用线条圈定这个色块
- 用户可自定义印章或水印图案
- 保存是增加作者印章 或 水印

## 终端兼容性

- 优先照顾移动端的体验
- 再是 PC 端/ Pad 端
